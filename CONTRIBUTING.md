# Contributing to English Coach

Thanks for your interest in improving English Coach.

Useful contribution types:
- clearer correction examples
- better onboarding / installation docs
- additional multilingual README improvements
- prompt behavior refinements that make corrections more helpful and less intrusive

Before opening a PR:
1. Describe the user problem clearly
2. Keep changes small and easy to review
3. If you change README or skill behavior, explain the expected user-facing impact

If you're unsure whether an idea fits, open an issue or discussion first.
